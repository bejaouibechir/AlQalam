## Objectif

Remplacer toutes les boucles `for` + `append` par des **compréhensions Python** (list, dict, set) et des **générateurs** (`yield`), et ajouter un onglet "Tableau de bord" avec KPIs, statistiques par catégorie, top 5 produits, et une recherche temps réel.

## Concepts mis en œuvre

| Expression             | Syntaxe                              | Remplace                       |
| ---------------------- | ------------------------------------ | ------------------------------ |
| **List comprehension** | `[x for x in iterable if condition]` | `for` + `append()`             |
| **Dict comprehension** | `{k: v for k, v in ...}`             | `for` + `dict[k] = v`          |
| **Set comprehension**  | `{expr for x in iterable}`           | `for` + `set.add()`            |
| **Générateur**         | `(expr for x in ...)` ou `yield`     | Liste intermédiaire en mémoire |

```python
def produits_en_alerte(self) -> list:

    for p in self._produits.values():
        if p.est_en_alerte():
            alertes.append(p)
    return alertes

# ✅ Après (list comprehension)
def produits_en_alerte(self) -> list:
    return [p for p in self._produits.values() if p.est_en_alerte()]
```

```python
# Filtre multi-champs : ref, nom OU catégorie contiennent le texte
def rechercher(self, texte: str) -> list:
    """
    [V3] List comprehension avec condition composée.
    Un seul parcours, pas de liste intermédiaire.
    """
    t = texte.lower().strip()
    if not t:
        return list(self._produits.values())   # tout retourner si vide
    return [
        p for p in self._produits.values()
        if t in p.ref.lower()
        or t in p.nom.lower()
        or t in p.categorie.lower()
    ]
```

### Étape 2 — Dict et Set comprehension : groupement (15 min)

```python
def par_categorie(self) -> dict:
    """
    Regroupe les produits par catégorie.
    [V3] Deux comprehensions imbriquées en deux lignes.
    """
    # Set comprehension : toutes les catégories uniques (sans doublon)
    categories = {p.categorie for p in self._produits.values()}

    # Dict comprehension : { catégorie → [liste de produits] }
    return {
        cat: [p for p in self._produits.values() if p.categorie == cat]
        for cat in sorted(categories)
    }
```

### Étape 3 — Dict comprehension avec générateur : statistiques (15 min)

```python
def stats_categories(self) -> dict:
    """
    Statistiques agrégées par catégorie.
    [V3] Dict comprehension imbriqué + expression génératrice dans sum().

    Retourne : { cat → { nb_produits, valeur_totale, nb_alertes } }
    """
    return {
        cat: {
            "nb_produits"  : len(prods),
            # Générateur dans sum() : pas de liste intermédiaire
            "valeur_totale": sum(p.valeur_stock() for p in prods),
            "nb_alertes"   : sum(1 for p in prods if p.est_en_alerte()),
        }
        for cat, prods in self.par_categorie().items()
    }
```

**Différence list vs générateur dans `sum()` :**

```python
produits = [p1, p2, p3, ...]  # 10 000 produits

# ❌ List comprehension : crée une liste de 10 000 floats EN MÉMOIRE
total = sum([p.valeur_stock() for p in produits])

# ✅ Expression génératrice : calcule valeur par valeur, sans liste
total = sum(p.valeur_stock() for p in produits)
#            ↑ pas de crochets → générateur, pas une liste
```

### Étape 4 — Générateur avec yield : export CSV (10 min)

```python
def flux_export(self):
    """
    Générateur de lignes CSV.
    [V3] yield : produit les données une par une sans tout charger en mémoire.

    Usage : for ligne in stock.flux_export(): csv_writer.writerow(ligne)
    """
    # Ligne d'en-tête
    yield ["ref", "nom", "categorie", "prix_achat", "prix_vente", "qte", "seuil_min"]

    # Une ligne par produit, dans l'ordre alphabétique (utilise __lt__ de V2)
    for p in sorted(self._produits.values()):
        yield [p.ref, p.nom, p.categorie,
               p.prix_achat, p.prix_vente, p.qte, p.seuil_min]
```

**Pourquoi `yield` au lieu de `return liste` ?**

```python
# ❌ Sans générateur : charge TOUT en mémoire avant de commencer à écrire
def exporter(self) -> list:
    lignes = [en_tete]
    for p in self._produits.values():
        lignes.append([p.ref, p.nom, ...])
    return lignes   # 10 000 produits → liste de 10 000 éléments en RAM

# ✅ Avec générateur : une ligne calculée, une ligne écrite, mémoire constante
for ligne in stock.flux_export():
    writer.writerow(ligne)   # calcule et écrit une ligne à la fois
```

```python
class StockFrame(ctk.CTkFrame):
    def _construire_filtres(self):
        # Champ de recherche texte
        self._var_recherche = ctk.StringVar()
        self._champ_recherche = ctk.CTkEntry(
            self, textvariable=self._var_recherche,
            placeholder_text="🔍 Rechercher (réf, nom, catégorie)…"
        )
        self._champ_recherche.pack(fill="x", padx=10, pady=5)
        # Déclenche la recherche à chaque frappe
        self._champ_recherche.bind("<KeyRelease>", lambda e: self.rafraichir())

        # Filtre catégorie (menu déroulant)
        self._var_categorie = ctk.StringVar(value="Toutes")
        self._menu_cat = ctk.CTkOptionMenu(
            self, variable=self._var_categorie,
            command=lambda _: self.rafraichir()
        )
        self._menu_cat.pack()

    def rafraichir(self):
        texte = self._var_recherche.get()
        cat   = self._var_categorie.get()

        # Appliquer les filtres (list comprehensions dans stock_service)
        produits = self._stock.rechercher(texte)
        if cat != "Toutes":
            produits = [p for p in produits if p.categorie == cat]

        # Mettre à jour le compteur
        self._lbl_compteur.configure(text=f"{len(produits)} produit(s) affiché(s)")

        # Remplir le tableau
        for item in self._arbre.get_children():
            self._arbre.delete(item)
        for p in produits:
            self._arbre.insert("", "end", iid=p.ref, values=(...))
```

## Récapitulatif des comprehensions utilisées

```python
# 1. LIST — filtrer
alertes = [p for p in stock if p.est_en_alerte()]

# 2. LIST — filtrer multi-conditions
resultats = [p for p in stock
             if texte in p.ref.lower() or texte in p.nom.lower()]

# 3. SET — valeurs uniques
categories = {p.categorie for p in stock}

# 4. DICT — grouper
par_cat = {cat: [p for p in stock if p.categorie == cat]
           for cat in categories}

# 5. DICT imbriqué — agréger
stats = {cat: {"nb": len(ps), "valeur": sum(p.valeur_stock() for p in ps)}
         for cat, ps in par_cat.items()}

# 6. GENERATOR — export mémoire-efficace
for ligne in (p.to_dict() for p in sorted(stock)):
    writer.writerow(ligne)
```
