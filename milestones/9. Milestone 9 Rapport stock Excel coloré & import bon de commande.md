## 1. Concepts clés — openpyxl

### 1.1 Pourquoi openpyxl ?

Le module standard Python ne gère pas Excel. `openpyxl` est la bibliothèque de référence pour lire et écrire des fichiers `.xlsx` (format Office Open XML). Elle ne nécessite pas que Microsoft Excel soit installé.

```bash
py -3 -m pip install openpyxl
```

Contrairement au module `csv` vu en V8, openpyxl manipule des **cellules typées** (nombres, dates, chaînes), gère les **styles visuels** et supporte les **graphiques intégrés**.

### 1.2 Workbook et Worksheet — la base

```python
from openpyxl import Workbook

# Créer un nouveau classeur (workbook)
wb = Workbook()

# La feuille active créée automatiquement
ws_active = wb.active
ws_active.title = "Feuille principale"

# Créer des feuilles supplémentaires
ws2 = wb.create_sheet(title="Statistiques")
ws3 = wb.create_sheet(title="Ruptures", index=1)  # index = position

# Écrire dans une cellule (deux syntaxes équivalentes)
ws_active["A1"] = "Référence"          # syntaxe dictionnaire
ws_active.cell(row=2, column=1, value="CRAY-001")  # syntaxe indicée

# Sauvegarder le classeur
wb.save("rapport_stock.xlsx")
print("Fichier Excel créé !")
```

**Points clés :**
- `Workbook()` crée toujours une feuille active par défaut
- `wb.create_sheet()` ajoute des onglets supplémentaires
- Les indices de lignes et colonnes commencent à **1** (pas 0)

### 1.3 PatternFill — colorisation des cellules

```python
from openpyxl.styles import PatternFill

# Définir des remplissages (fill) avec des couleurs hexadécimales RRGGBB
fill_rouge   = PatternFill(fill_type="solid", fgColor="FF4444")  # rupture
fill_orange  = PatternFill(fill_type="solid", fgColor="FFA500")  # alerte
fill_vert    = PatternFill(fill_type="solid", fgColor="44BB44")  # normal
fill_entete  = PatternFill(fill_type="solid", fgColor="1F3864")  # bleu nuit

# Appliquer un remplissage à une cellule
cellule = ws.cell(row=3, column=1, value="CRAY-001")
cellule.fill = fill_rouge

# Colorisation conditionnelle selon la quantité
def choisir_fill(qte, seuil_min):
    if qte <= 0:
        return fill_rouge        # rupture : rouge
    elif qte <= seuil_min:
        return fill_orange       # alerte  : orange
    else:
        return fill_vert         # normal  : vert
```

**Tableau des couleurs utilisées dans Al Qalam :**

| État | Couleur | Code hex | Description |
|------|---------|----------|-------------|
| Rupture | Rouge | `FF4444` | `qte <= 0` |
| Alerte | Orange | `FFA500` | `0 < qte <= seuil_min` |
| Normal | Vert | `44BB44` | `qte > seuil_min` |
| En-tête | Bleu nuit | `1F3864` | Ligne de titre |
| Total | Gris | `D9D9D9` | Ligne de synthèse |

### 1.11 BarChart — graphique en barres

```python
from openpyxl.chart import BarChart, Reference

# Supposons que la feuille "Statistiques" contient :
# Colonne A : noms de catégories (A2:A6)
# Colonne B : quantités totales (B2:B6)

ws_stats = wb["Statistiques"]

# Référence aux données (sans l'en-tête)
donnees = Reference(
    ws_stats,
    min_col=2,      # colonne B = index 2
    min_row=1,      # inclure la ligne d'en-tête pour le titre de série
    max_row=6       # jusqu'à la ligne 6
)

# Référence aux étiquettes de l'axe X (catégories)
categories = Reference(
    ws_stats,
    min_col=1,      # colonne A
    min_row=2,      # commencer après l'en-tête
    max_row=6
)

# Créer le graphique
chart = BarChart()
chart.type = "col"           # "col" = barres verticales, "bar" = horizontales
chart.title = "Stock par catégorie"
chart.y_axis.title = "Quantité"
chart.x_axis.title = "Catégorie"
chart.style = 10             # style visuel prédéfini (1-48)
chart.width = 20             # largeur en cm
chart.height = 12            # hauteur en cm

# Associer les données et les étiquettes
chart.add_data(donnees, titles_from_data=True)
chart.set_categories(categories)

# Insérer le graphique dans la feuille à partir de la cellule A15
ws_stats.add_chart(chart, "A15")
```

### 3.2 ExcelService — export du rapport

```python
# services/excel_service.py

from __future__ import annotations
from pathlib import Path
from datetime import datetime
from openpyxl import Workbook, load_workbook
from openpyxl.styles import PatternFill, Font, Alignment, Border, Side
from openpyxl.utils import get_column_letter
from openpyxl.chart import BarChart, Reference

from config import (
    RAPPORT_EXCEL_CHEMIN, EXCEL_DIR,
    EXCEL_COULEUR_RUPTURE, EXCEL_COULEUR_ALERTE,
    EXCEL_COULEUR_NORMAL, EXCEL_COULEUR_ENTETE, EXCEL_COULEUR_TOTAL
)


class ExcelService:
    """Service responsable de toutes les opérations Excel (openpyxl)."""

    # ------------------------------------------------------------------
    # Styles réutilisables — définis une seule fois comme attributs de classe
    # ------------------------------------------------------------------
    FILL_RUPTURE = PatternFill(fill_type="solid", fgColor=EXCEL_COULEUR_RUPTURE)
    FILL_ALERTE  = PatternFill(fill_type="solid", fgColor=EXCEL_COULEUR_ALERTE)
    FILL_NORMAL  = PatternFill(fill_type="solid", fgColor=EXCEL_COULEUR_NORMAL)
    FILL_ENTETE  = PatternFill(fill_type="solid", fgColor=EXCEL_COULEUR_ENTETE)
    FILL_TOTAL   = PatternFill(fill_type="solid", fgColor=EXCEL_COULEUR_TOTAL)

    FONT_TITRE   = Font(bold=True, color="1F3864", size=14, name="Calibri")
    FONT_ENTETE  = Font(bold=True, color="FFFFFF", size=11, name="Calibri")
    FONT_NORMAL  = Font(size=10, name="Calibri")
    FONT_TOTAL   = Font(bold=True, size=10, name="Calibri")

    ALIGN_CENTRE = Alignment(horizontal="center", vertical="center", wrap_text=True)
    ALIGN_GAUCHE = Alignment(horizontal="left",   vertical="center")
    ALIGN_DROITE = Alignment(horizontal="right",  vertical="center")

    COTE_FIN     = Side(style="thin")

    @classmethod
    def _bordure_fine(cls) -> Border:
        """Retourne une bordure fine sur les 4 côtés."""
        return Border(
            left=cls.COTE_FIN, right=cls.COTE_FIN,
            top=cls.COTE_FIN,  bottom=cls.COTE_FIN
        )

    @classmethod
    def _choisir_fill(cls, qte: int, seuil_min: int) -> PatternFill:
        """Colorisation conditionnelle selon la quantité en stock."""
        if qte <= 0:
            return cls.FILL_RUPTURE
        elif qte <= seuil_min:
            return cls.FILL_ALERTE
        else:
            return cls.FILL_NORMAL

    # ------------------------------------------------------------------
    # Export principal : rapport multi-feuilles
    # ------------------------------------------------------------------
    @classmethod
    def exporter_rapport_stock(cls, produits: list) -> Path:
        """
        Génère un classeur Excel multi-feuilles avec :
          - Feuille 1 : Catalogue complet colorisé
          - Feuille 2 : Ruptures et alertes seulement
          - Feuille 3 : Statistiques par catégorie + graphique

        Retourne le chemin du fichier créé.
        """
        wb = Workbook()   # classeur vide (crée une feuille "Sheet" par défaut)

        # Supprimer la feuille vide créée par défaut
        del wb["Sheet"]

        # Créer les 3 feuilles dans l'ordre souhaité
        ws_catalogue = wb.create_sheet("📦 Catalogue")
        ws_ruptures  = wb.create_sheet("⚠️ Ruptures")
        ws_stats     = wb.create_sheet("📊 Statistiques")

        # Remplir chaque feuille
        cls._remplir_catalogue(ws_catalogue, produits)
        cls._remplir_ruptures(ws_ruptures, produits)
        cls._remplir_statistiques(ws_stats, produits)

        # Sauvegarder le fichier
        RAPPORT_EXCEL_CHEMIN.parent.mkdir(parents=True, exist_ok=True)
        wb.save(RAPPORT_EXCEL_CHEMIN)

        return RAPPORT_EXCEL_CHEMIN

    # ------------------------------------------------------------------
    # Feuille 1 : Catalogue complet
    # ------------------------------------------------------------------
    @classmethod
    def _remplir_catalogue(cls, ws, produits: list) -> None:
        """Construit la feuille 'Catalogue' avec colorisation conditionnelle."""

        # --- Ligne 1 : Titre fusionné ---
        ws.merge_cells("A1:G1")    # fusionner de A à G (7 colonnes)
        titre = ws["A1"]
        titre.value     = "📦 Catalogue Al Qalam — Rapport de stock"
        titre.font      = Font(bold=True, color="FFFFFF", size=14, name="Calibri")
        titre.fill      = cls.FILL_ENTETE
        titre.alignment = cls.ALIGN_CENTRE
        ws.row_dimensions[1].height = 35   # hauteur de la ligne titre

        # --- Ligne 2 : Date de génération ---
        ws.merge_cells("A2:G2")
        ws["A2"].value     = f"Généré le {datetime.now().strftime('%d/%m/%Y à %H:%M')}"
        ws["A2"].font      = Font(italic=True, size=9, color="666666")
        ws["A2"].alignment = Alignment(horizontal="right")

        # --- Ligne 3 : En-têtes des colonnes ---
        entetes = ["Référence", "Nom du produit", "Catégorie", "Quantité", "Seuil min", "Prix (DT)", "État"]
        largeurs = [12, 30, 15, 11, 11, 12, 12]

        for col_idx, (entete, largeur) in enumerate(zip(entetes, largeurs), start=1):
            lettre = get_column_letter(col_idx)          # 1 → "A", 2 → "B"...
            ws.column_dimensions[lettre].width = largeur  # largeur de colonne

            cellule = ws.cell(row=3, column=col_idx, value=entete)
            cellule.font      = cls.FONT_ENTETE
            cellule.fill      = cls.FILL_ENTETE
            cellule.alignment = cls.ALIGN_CENTRE
            cellule.border    = cls._bordure_fine()

        ws.row_dimensions[3].height = 22   # hauteur ligne en-tête

        # Figer les 3 premières lignes (titre + date + en-têtes)
        ws.freeze_panes = "A4"

        # --- Lignes de données (à partir de la ligne 4) ---
        total_valeur = 0.0
        total_qte    = 0

        for ligne_idx, produit in enumerate(produits, start=4):
            qte       = produit.quantite
            seuil_min = produit.seuil_min
            fill_etat = cls._choisir_fill(qte, seuil_min)

            # Déterminer le libellé de l'état
            if qte <= 0:
                etat = "🔴 Rupture"
            elif qte <= seuil_min:
                etat = "🟠 Alerte"
            else:
                etat = "🟢 Normal"

            # Données de la ligne
            valeurs = [
                produit.reference,
                produit.nom,
                produit.categorie,
                qte,
                seuil_min,
                produit.prix_unitaire,
                etat,
            ]

            for col_idx, valeur in enumerate(valeurs, start=1):
                cellule = ws.cell(row=ligne_idx, column=col_idx, value=valeur)
                cellule.border    = cls._bordure_fine()
                cellule.font      = cls.FONT_NORMAL
                cellule.alignment = cls.ALIGN_CENTRE if col_idx != 2 else cls.ALIGN_GAUCHE

            # Coloriser UNIQUEMENT la colonne "Quantité" (colonne 4)
            ws.cell(row=ligne_idx, column=4).fill = fill_etat

            # Accumuler pour la ligne de total
            total_qte    += qte
            total_valeur += qte * produit.prix_unitaire

            ws.row_dimensions[ligne_idx].height = 18

        # --- Ligne de total ---
        ligne_total = len(produits) + 4
        ws.merge_cells(f"A{ligne_total}:C{ligne_total}")
        ws[f"A{ligne_total}"].value     = "TOTAL"
        ws[f"A{ligne_total}"].font      = cls.FONT_TOTAL
        ws[f"A{ligne_total}"].fill      = cls.FILL_TOTAL
        ws[f"A{ligne_total}"].alignment = cls.ALIGN_CENTRE

        ws.cell(row=ligne_total, column=4, value=total_qte).fill   = cls.FILL_TOTAL
        ws.cell(row=ligne_total, column=6, value=round(total_valeur, 3)).fill = cls.FILL_TOTAL

        for col_idx in range(1, 8):
            ws.cell(row=ligne_total, column=col_idx).font   = cls.FONT_TOTAL
            ws.cell(row=ligne_total, column=col_idx).border = cls._bordure_fine()

    # ------------------------------------------------------------------
    # Feuille 2 : Ruptures et alertes
    # ------------------------------------------------------------------
    @classmethod
    def _remplir_ruptures(cls, ws, produits: list) -> None:
        """Feuille ne contenant que les produits en rupture ou en alerte."""

        # Filtrer les produits problématiques
        en_alerte = [p for p in produits if p.quantite <= p.seuil_min]

        # Titre
        ws.merge_cells("A1:F1")
        ws["A1"].value     = "⚠️ Produits en rupture ou en alerte"
        ws["A1"].font      = Font(bold=True, color="FFFFFF", size=13)
        ws["A1"].fill      = PatternFill(fill_type="solid", fgColor="C00000")
        ws["A1"].alignment = cls.ALIGN_CENTRE
        ws.row_dimensions[1].height = 30

        # En-têtes
        entetes = ["Référence", "Nom", "Catégorie", "Quantité", "Seuil min", "À commander"]
        for col_idx, entete in enumerate(entetes, start=1):
            c = ws.cell(row=2, column=col_idx, value=entete)
            c.font      = cls.FONT_ENTETE
            c.fill      = cls.FILL_ENTETE
            c.alignment = cls.ALIGN_CENTRE
            c.border    = cls._bordure_fine()

        ws.freeze_panes = "A3"

        # Données
        for ligne_idx, produit in enumerate(en_alerte, start=3):
            a_commander = max(0, produit.seuil_min * 2 - produit.quantite)  # suggestion
            valeurs = [
                produit.reference, produit.nom, produit.categorie,
                produit.quantite, produit.seuil_min, a_commander
            ]
            fill = cls.FILL_RUPTURE if produit.quantite <= 0 else cls.FILL_ALERTE

            for col_idx, valeur in enumerate(valeurs, start=1):
                c = ws.cell(row=ligne_idx, column=col_idx, value=valeur)
                c.border    = cls._bordure_fine()
                c.alignment = cls.ALIGN_CENTRE
                c.font      = cls.FONT_NORMAL
                if col_idx == 4:   # colonne Quantité uniquement
                    c.fill = fill

        # Largeurs
        for lettre, largeur in zip(["A","B","C","D","E","F"], [12, 28, 14, 11, 11, 13]):
            ws.column_dimensions[lettre].width = largeur

    # ------------------------------------------------------------------
    # Feuille 3 : Statistiques par catégorie + graphique
    # ------------------------------------------------------------------
    @classmethod
    def _remplir_statistiques(cls, ws, produits: list) -> None:
        """Agrège les données par catégorie et insère un graphique en barres."""

        # --- Agrégation par catégorie ---
        stats: dict[str, dict] = {}
        for p in produits:
            cat = p.categorie
            if cat not in stats:
                stats[cat] = {"nb_produits": 0, "qte_totale": 0, "valeur_totale": 0.0}
            stats[cat]["nb_produits"]  += 1
            stats[cat]["qte_totale"]   += p.quantite
            stats[cat]["valeur_totale"] += p.quantite * p.prix_unitaire

        # --- Titre ---
        ws.merge_cells("A1:E1")
        ws["A1"].value     = "📊 Statistiques par catégorie"
        ws["A1"].font      = Font(bold=True, color="FFFFFF", size=13)
        ws["A1"].fill      = cls.FILL_ENTETE
        ws["A1"].alignment = cls.ALIGN_CENTRE
        ws.row_dimensions[1].height = 30

        # --- En-têtes ---
        entetes = ["Catégorie", "Nb produits", "Qté totale", "Valeur (DT)", "% du stock"]
        for col_idx, entete in enumerate(entetes, start=1):
            c = ws.cell(row=2, column=col_idx, value=entete)
            c.font      = cls.FONT_ENTETE
            c.fill      = cls.FILL_ENTETE
            c.alignment = cls.ALIGN_CENTRE
            c.border    = cls._bordure_fine()

        ws.freeze_panes = "A3"

        # --- Données agrégées ---
        valeur_totale_globale = sum(s["valeur_totale"] for s in stats.values()) or 1

        for ligne_idx, (categorie, data) in enumerate(stats.items(), start=3):
            pct = round(data["valeur_totale"] / valeur_totale_globale * 100, 1)
            valeurs = [
                categorie,
                data["nb_produits"],
                data["qte_totale"],
                round(data["valeur_totale"], 3),
                f"{pct} %"
            ]
            for col_idx, valeur in enumerate(valeurs, start=1):
                c = ws.cell(row=ligne_idx, column=col_idx, value=valeur)
                c.border    = cls._bordure_fine()
                c.alignment = cls.ALIGN_CENTRE
                c.font      = cls.FONT_NORMAL

        # --- Graphique en barres (quantité totale par catégorie) ---
        nb_cats   = len(stats)
        derniere  = nb_cats + 2    # dernière ligne de données

        # Données : colonne C (Qté totale), lignes 2 à derniere
        donnees = Reference(ws, min_col=3, min_row=2, max_row=derniere)

        # Étiquettes : colonne A (Catégorie), lignes 3 à derniere
        categories_ref = Reference(ws, min_col=1, min_row=3, max_row=derniere)

        chart = BarChart()
        chart.type        = "col"                    # barres verticales
        chart.title       = "Quantité en stock par catégorie"
        chart.y_axis.title = "Quantité"
        chart.x_axis.title = "Catégorie"
        chart.style       = 10                       # style visuel prédéfini
        chart.width       = 20                       # largeur en cm
        chart.height      = 12                       # hauteur en cm

        chart.add_data(donnees, titles_from_data=True)
        chart.set_categories(categories_ref)

        # Positionner le graphique sous le tableau (ligne 15 environ)
        ws.add_chart(chart, "A15")

        # Largeurs de colonnes
        for lettre, largeur in zip(["A","B","C","D","E"], [20, 13, 12, 14, 12]):
            ws.column_dimensions[lettre].width = largeur

    # ------------------------------------------------------------------
    # Import : bon de commande fournisseur
    # ------------------------------------------------------------------
    @classmethod
    def importer_bon_commande(cls, chemin: str, stock_service) -> dict:
        """
        Lit un fichier Excel fournisseur et appelle entree_stock()
        pour chaque ligne valide.

        Format attendu du fichier fournisseur :
          Ligne 1 : en-têtes (référence, quantite, note)
          Lignes 2+ : données

        Retourne un rapport : {"importes": int, "erreurs": list, "total_lignes": int}
        """
        rapport = {"importes": 0, "erreurs": [], "total_lignes": 0}

        # Ouvrir le fichier en lecture seule avec les valeurs calculées
        wb = load_workbook(chemin, data_only=True)
        ws = wb.active   # utiliser la première feuille

        lignes = list(ws.iter_rows(values_only=True))
        if not lignes:
            rapport["erreurs"].append("Fichier vide")
            return rapport

        # La première ligne est l'en-tête — on la saute
        entetes = [str(e).strip().lower() if e else "" for e in lignes[0]]
        donnees = lignes[1:]
        rapport["total_lignes"] = len(donnees)

        for num_ligne, ligne in enumerate(donnees, start=2):
            try:
                # Ignorer les lignes entièrement vides
                if all(v is None for v in ligne):
                    continue

                ref  = str(ligne[0]).strip() if ligne[0] is not None else ""
                qte  = int(ligne[1])              if ligne[1] is not None else 0
                note = str(ligne[2]).strip()      if len(ligne) > 2 and ligne[2] is not None else ""

                if not ref:
                    raise ValueError("Référence manquante")
                if qte <= 0:
                    raise ValueError(f"Quantité invalide : {qte}")

                # Appeler le service stock pour enregistrer l'entrée
                stock_service.entree_stock(ref, qte, note or f"Import BC ligne {num_ligne}")
                rapport["importes"] += 1

            except Exception as exc:
                rapport["erreurs"].append(f"Ligne {num_ligne} : {exc}")

        return rapport

    # ------------------------------------------------------------------
    # Prévisualisation : lecture partielle pour l'UI
    # ------------------------------------------------------------------
    @classmethod
    def previsualiser_excel(cls, chemin: str, max_lignes: int = 10) -> dict:
        """
        Lit les premières lignes d'un fichier Excel pour affichage dans
        un Treeview (sans importer les données dans le stock).

        Retourne : {"entetes": list, "lignes": list[tuple]}
        """
        wb  = load_workbook(chemin, data_only=True, read_only=True)
        ws  = wb.active
        res = {"entetes": [], "lignes": []}

        for i, ligne in enumerate(ws.iter_rows(values_only=True)):
            if i == 0:
                res["entetes"] = [str(v) if v is not None else "" for v in ligne]
            elif i <= max_lignes:
                res["lignes"].append(tuple(
                    str(v) if v is not None else "" for v in ligne
                ))
            else:
                break

        wb.close()
        return res
```

```python
        textbox.insert("end", message)
        textbox.configure(state="disabled")
```

### Ce qui a évolué entre V8 et V9

| Aspect | V8 (CSV) | V9 (Excel) |
|--------|----------|------------|
| Module | `csv` (stdlib) | `openpyxl` (tiers) |
| Format de fichier | `.csv` (texte brut) | `.xlsx` (binaire ZIP) |
| Feuilles multiples | Non | Oui (`create_sheet`) |
| Styles visuels | Non | Oui (couleurs, polices, bordures) |
| Graphiques | Non | Oui (`BarChart`) |
| Formules Excel | Non | Possible (V10+) |
| Encodage | `utf-8-sig` obligatoire | Natif (pas d'encodage) |
| Lecture | `DictReader` | `iter_rows(values_only=True)` |

### Points d'attention

- **Toujours appeler `wb.save()`** après vos modifications — sinon rien n'est écrit sur le disque.
- **`data_only=True`** dans `load_workbook` retourne les valeurs calculées au lieu des formules Excel brutes.
- **`read_only=True`** dans `load_workbook` améliore les performances sur les gros fichiers — mais interdit les modifications. N'oubliez pas `wb.close()`.
- Les indices de lignes et colonnes commencent à **1** dans openpyxl (contrairement aux listes Python qui commencent à 0).
- `PatternFill` sans `fill_type="solid"` n'applique pas de couleur — c'est un oubli fréquent.
- Les cellules fusionnées avec `merge_cells` : n'écrire que dans la **cellule supérieure gauche** de la plage.
